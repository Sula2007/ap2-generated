# ap2-generated

Auto-generated Go gRPC code from [ap2-protos](https://github.com/Sula2007/ap2-protos).

Generated automatically by GitHub Actions on every push to the proto repository.

## Install

```bash
go get github.com/Sula2007/ap2-generated@v1.0.0
```

## Packages

- `github.com/Sula2007/ap2-generated/payment` — PaymentService gRPC
- `github.com/Sula2007/ap2-generated/order` — OrderService streaming gRPC
